# Products Inventory Web Application

This is a simple PHP-based Product Inventory web application that allows users to view a list of products and add new products through a web form. The project is designed for educational purposes to demonstrate PHP form handling, Docker containerization, and deployment to a VPS environment.

## Features
- Display a list of products in a table
- Add new products using a form
- Basic form validation
- Styled using Bootstrap
- Fully containerized using Docker

## Technologies Used
- PHP 8.2
- Apache Web Server
- Docker
- Git & GitHub
- Ubuntu VPS
